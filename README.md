# Fake News Detector v1

This branch preserves the early prototype before the final dual-dimension misinformation analyzer was consolidated.

## Included

- `backend/fact_checking/v1`: first fact-checking pipeline prototype.
- `backend/fact_checking/v2`: improved retrieval, decision logic, evaluation files, and experiment notebooks.
- `backend/fake_news_text_feature`: early text-feature and model-training experiments, excluding copied weights.
- `frontend`: early frontend prototype.
- `dataset`: datasets used by the early prototype.

## Excluded

Generated Python caches, model checkpoints, copied model weights, and large training archives are intentionally excluded from this archive branch.

The final consolidated implementation is maintained on the repository `master` branch.
