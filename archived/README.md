# Archived Work

This folder preserves earlier project material that supports the development history of the COMP3000 project.

## Contents

- `previous work/`: early Colab notebooks, machine learning experiments, result tables, figures, and related research work.

## Excluded Material

Large model checkpoints, binary model weights, Python caches, and compressed archive exports are intentionally excluded from version control.

The archive is included for traceability and reflection rather than as the runnable final application. The runnable final implementation is in `dual_dimension_misinformation_analyzer_final`.
